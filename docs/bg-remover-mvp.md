# Image Background Remover - MVP 需求文档

## 1. 产品概述

| 项目 | 说明 |
|------|------|
| 产品名称 | BG Remover |
| 产品定位 | 轻量级在线图片背景移除工具 |
| 核心价值 | 快速、免费、无需安装 |

---

## 2. 功能需求

### 2.1 核心功能

| 优先级 | 功能 | 描述 |
|:------:|------|------|
| P0 | 图片上传 | 支持拖拽或点击上传（PNG/JPG，最大 10MB） |
| P0 | 背景移除 | 一键移除图片背景 |
| P0 | 结果预览 | 展示原图 vs 移除背景后的对比 |
| P0 | 图片下载 | 下载透明背景 PNG |
| P1 | 使用次数限制 | 免费用户每天 3 次（可选） |

### 2.2 用户流程

```
1. 访问网站
2. 拖拽/选择图片
3. 自动处理（显示 Loading）
4. 展示对比结果
5. 下载结果
```

---

## 3. 技术方案

### 3.1 架构

```
┌─────────────┐     ┌─────────────┐     ┌──────────┐
│  Cloudflare │ ──▶ │ Cloudflare │ ──▶ │Remove.bg │
│   Pages     │     │  Workers   │     │   API    │
└─────────────┘     └─────────────┘     └──────────┘
   (前端页面)         (后端 API)          (图片处理)
```

### 3.2 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端 | HTML + CSS + JavaScript（原生） |
| 后端 | Cloudflare Workers |
| API | Remove.bg |
| 部署 | Cloudflare（免费额度内） |

### 3.3 成本估算

| 项目 | 免费额度 | 超出费用 |
|------|----------|----------|
| Cloudflare Workers | 100,000 次/天 | - |
| Cloudflare 流出带宽 | 1GB/天 | $0.15/GB |
| Remove.bg | 50 次/月 | $0.019/张 |

## 5. 接口设计

### 5.1 后端 API

**POST /api/remove-bg**

| 参数 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| image | File | ✅ | 图片文件 |

**响应**
```json
{
  "success": true,
  "image_url": "data:image/png;base64,...",
  "credits_remaining": 48
}
```
